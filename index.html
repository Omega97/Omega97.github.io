Mini-Hold'Em (Coursework Project)

In this project, I tested how evolutionary algorithms perform in mastering hidden information card games of various degrees of complexity. Mini-Hold'em is a simplified version of Poker Texas Hold'Em. The bot is able to achieve human-level play for simpler versions of the game, but it struggles when variability is too high.

Mancala Self-Reinforcement Learning AI (Coursework Project)

I created a self-reinforcement learning AI for the game of Mancala. The agent uses MCTS to choose the best move. A neural-network-based policy is used to guide the search. Roll-outs are performed to evaluate the leaf nodes. The roll-outs are truncated and evaluated with a value network. Both the value net and the policy are trained on self-play data, generated with the help of a custom Mancala simulator that I created.

Wiki-Tok (Coursework Project)

This group project implements a recommendation system of Wikipedia pages. The user can provide feedback for each page, and then he is fed with the next one, just like with the app TikTok.

Search Heuristics (Independent Study)

In the rich and well-studied field of optimization, one generic use case is to find the best one among a set of random variables, trading off between reward and exploration cost. This small project provides a simple implementation and comparison of some of the best-performing heuristics, ready to be applied in real-world scenarios.

Quantum Logic Gates

In this project, I tested my knowledge of quantum mechanics by creating a repository that contains a library for simulating quantum logic gates. The library enables the user to visualize the process of applying quantum operators and perform simple quantum algorithms.

Steganography (Independent Study)

Neural Networks from Scratch (Independent Project)

Ramanujan Machine (Independent Project)

The Ramanujan Machine is a tool based on continued fractions and gradient descent. It allows finding interesting equations for important mathematical constants. I created a GitHub repository that successfully implemented this tool, and it got featured in a YouTube video.

Quantum Ground State Simulation (Independent Study)

Using my knowledge of quantum mechanics, I created a repository that allows you to simulate the ground state of a quantum system of interacting particles. Given the potential, the program will find the ground state of the system using the variational method. The program makes the distinction between bosons and fermions. It can also handle potentials with a singularity at the origin.

Wave simulation (Independent Project)

I created a simple simulation of a 2D wave using the Runge-Kutta method. The library allows for different boundary conditions, initial conditions, and obstacles. The simulation can be exported to a GIF.
